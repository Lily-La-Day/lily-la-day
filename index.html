    Lily Day

    Lily Day is a full-stack software developer. She has most recently been employed by VMware and has been working on a SaaS cloud analytics platform called CloudHealth. She is fortunate enough to have gained a wide range of experience - from Ruby on Rails to TypeScript with React to Kubernetes and GraphQL. A graduate of "code school" (General Assembly Software Engineering Immersive), she still can't quite believe her luck to have found something she can do professionally that she enjoys so much.

    July 2019

    • Am i write? - Full Stack - Flask/Python with React.js

      Am i write? is an online platform that allows two distinct user types to either submit writing to be edited or to edit the writing of others. The application’s backend was created using Python and Flask with a React.js front-end. It consumes the Words API which allows for the automatic generation of synonyms and definitions of words as a new edit is written.
      Deployed: 15 Aug 2019 579

    • Stranger Things - Full Stack - MERN

      A full-stack MERN (Moongoose, Express, React, Node.js) app developed as part of a group over one week. Users can view events filtered by tagged category and view users whose interests match these tags they can then request to be accepted into the event. They will then be alerted by private message when a request is accepted. Once accepted they can post on the event pages themselves and see private details of the events. Can send private messages to other users and receive messages themselves. They can also create their own event. As an event "host" they will receive notifications of interest from other users and can either accept or decline these requests.
      Deployed: July 2019 240
